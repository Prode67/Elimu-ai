# 📚 ElimuAI — Msaidizi wa Masomo

AI-powered education assistant for O Level & A Level students in Tanzania.

## 🚀 Deploy kwa Vercel (Dakika 10)

### Hatua 1: Weka GitHub
1. Nenda [github.com](https://github.com) → fungua account
2. Bonyeza **"New repository"**
3. Jina: `elimu-ai` → bonyeza **"Create repository"**
4. Upload files zote za folder hii

### Hatua 2: Deploy Vercel
1. Nenda [vercel.com](https://vercel.com) → Sign up na GitHub
2. Bonyeza **"New Project"**
3. Chagua repo yako `elimu-ai`
4. Bonyeza **"Deploy"** — tayari! 🎉

### Hatua 3: Shiriki Link
- Vercel itakupa link kama: `https://elimu-ai-xyz.vercel.app`
- Shiriki link hiyo na wanafunzi!

## 🔑 API Key
Kila mtumiaji ataingiza API key yake mwenyewe wakati wa kwanza kutumia app.
- Pata bure: [console.anthropic.com](https://console.anthropic.com)
